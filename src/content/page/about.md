This POC exists to answer one question: can a non-technical editor change the title, body text, and hero image of a page using only the Tina editor in a browser, and see those edits appear on the live site without touching any code, terminal, or git.

If you are reading this on the deployed URL, the site build works. If you are reading it after edits in the Tina editor, the loop works.
